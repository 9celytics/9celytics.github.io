Portfolio

This portfolio contains projects on Excel, SQL, Tableau, PowerBI and Python. This tools were used to Query, Manipulate and visualize data. Please click on each project for more.

SALES INSIGHT DASHBOARD

The dashboard shows insight on sales done in Nigeria and South Africa, the data contains 8905 rows. The data contains customers demographics, shipping status, sales producers, outlet type and a lot more.

SQL HACKERRANK PROJECT

This project contains solved solutions from problems on Hackerrank.com. The project involved querying over 20 questions on Hackerrank ranging from basic to intermediate questions.

Prosper Loan Analysis

This data set contains 113,937 loans with 81 variables on each loan, including loan amount, borrower rate (or interest rate), current loan status, borrower income, and many others.

KMI Superstore Analysis

The dashboard shows insight on KMI Superstore. I visualized the data to show percentage sales by customer segment, sales by province, sales by the product category and profit generated in quarters.

PWC CUSTOMER RETENTION DASHBOARD

The data provided by PWC company, here I checked for the trend on how people are leaving the product for other products, why they are doing so to give recommendations on how to improve our customer retention.

Bike Sales Analysis

In this project, I evaluated the bike sales data to understand different factors that are considered by the customers before making a purchase decision.
